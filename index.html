CSS Selectors & Specificity Cheat Sheet

Selectors

Selector Denotation Description CSS Example
Type [HTML element]{} Selects all matching elements. h1 {}
Class .[class name]{} Selects all elements with matching class name. .bold {}
ID #[id name]{} Selects the element with the matching ID name. #socials {}
Grouped [slector], [selector]{} Selectors can be grouped using a comma-separated list. In the example the CSS rule will apply to h1, h2 and h3 elements. h1, h2, h3 {}
Chained [slector][selector]{} Selects all elements that match all chained criteria. In the example only h3 elements that also have the section-heading class will be selected. h3.section-heading {}
Descendant [slector] [selector]{} Selects all elements that are descended from another matched selector. In the example only p elements inside the footer element will be selected. footer p {}

Specificity

Specificity is a ranking system used when there are multiple conflicting property values that point to the same element. When determining which rule to apply, the selector with the highest specificity wins.

Specificity ranking, from most specific to least specific:

ID selector > Class selectors > Type selectors
